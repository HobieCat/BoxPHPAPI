## Features
* Fetches the user details
* Folder:
	* List the items
	* List the collaborators
	* List the folders/files
	* Update folder details
	* Create Folder
	* Delete Folder & its content
	* Share Folder
* Files:
	* Fetches the file details
	* Upload file
	* Update file details
	* Delete file
	* Share file

## Install

Via Composer

``` bash
$ composer require hobiecat/boxphpapi
```
